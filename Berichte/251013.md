# Host Discovery

## Ping Script

### Funktionsweise
Es wird jede Adresse gepingt. Da dies sehr viele Adressen sein können, wird nur die Ping Ausgabe bei Erfolg ausgegeben.

### Script
```bash
#!/bin/bash

SUBNET="192.168.1"

for i in {1..254}
do
    IP="${SUBNET}.${i}"

    if ping -c 3 -W 1 "$IP" > /dev/null 2>&1; then
        ping -c 3 -W 1 "$IP"
    fi
done
```
